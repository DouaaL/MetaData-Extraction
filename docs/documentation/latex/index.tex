\hypertarget{index_autotoc_md3}{}\doxysection{\texorpdfstring{2. Présentation générale du projet}{2. Présentation générale du projet}}\label{index_autotoc_md3}
Ce projet a pour objectif d’implémenter une application Python capable de \+:


\begin{DoxyItemize}
\item lire des fichiers audio (MP3 / FLAC),
\item extraire et afficher leurs métadonnées,
\item modifier les tags directement dans les fichiers,
\item parcourir un dossier audio complet,
\item générer une playlist au format XSPF,
\item lire les playlists en mode interactif (piste suivante / précédente),
\item offrir une interface graphique (GUI) et inerface console (CLI).
\end{DoxyItemize}

Le projet est construit selon une {\bfseries{architecture modulaire}}, séparant \+:


\begin{DoxyItemize}
\item {\bfseries{l’interface utilisateur (CLI et GUI)}}, ~\newline

\item {\bfseries{la logique métier}} (playlist, bibliothèque audio), ~\newline

\item {\bfseries{la représentation des fichiers audio}} (MP3, FLAC, etc.).
\end{DoxyItemize}

Cette organisation rend le projet plus clair, maintenable et évolutif.
